## 1. 핵심 화면 와이어프레임

> 모든 와이어프레임은 360px 너비 기준 모바일 화면을 표현합니다.
> `[  ]` = 버튼/탭 가능한 요소, `####` = 이미지/차트 플레이스홀더

### 3.1 스와이프

**운세 카테고리 카드 스와이프:**
```
← ──────────────────────────────── →
│ [연애운 카드] [재물운 카드] [건강운] │

```
- 좌우 스와이프로 운세 카테고리 탐색
- 스냅 스크롤 (`scroll-snap-type: x mandatory`)
- 현재 위치 인디케이터 (점)

**채팅 메시지 스와이프 (롱프레스 대안):**
- AI 메시지 위에서 오른쪽으로 스와이프 → "공유" 아이콘 노출
- 스와이프 거리 비례 불투명도 애니메이션

### 3.2 풀다운 리프레시

**오늘의 운세 갱신:**
```
┌─────────────────────────────────────┐

└─────────────────────────────────────┘
```

- 당기기 80px: "새로고침 중..." 텍스트 표시
- 손 뗌: 실제 API 호출 → 완료 시 체크마크 스냅

### 3.5 토스트 / 알림 패턴

**토스트 분류 및 위치:**

```
┌─────────────────────────────────────┐

│ └─────────────────────────────────┘ │
│                                     │
│                  [화면 콘텐츠]       │
│                                     │
│ ┌─────────────────────────────────┐ │  ← 에러 토스트 (상단, 빨강)

│ └─────────────────────────────────┘ │
│                                     │
│ ┌─────────────────────────────────┐ │  ← 정보 토스트 (상단, 파랑)

│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**토스트 동작 규칙:**
- 진입: 상단에서 슬라이드 다운 (200ms ease-out)
- 퇴장: 위로 슬라이드 업 (150ms ease-in) 또는 페이드 아웃
- 자동 사라짐: 성공/정보 3초, 경고 5초, 에러 수동 닫기
- 최대 동시 표시: 2개 (스택)
- `sonner` 라이브러리 사용 권장

### 3.6 Haptic Feedback 패턴

| 상황 | 진동 패턴 | 비고 |
|------|----------|------|
| 사주 분석 완료 | 중간 진동 1회 (50ms) | iOS: `impactFeedback.medium` |
| 궁합 결과 로드 | 중간 진동 1회 | 결과 진입 시 |
| 버튼 탭 | 약한 진동 (20ms) | 중요 CTA만 |
| 드럼롤 값 변경 | 미세 진동 (10ms) | 날짜 피커 |
| 에러 발생 | 강한 진동 + 간격 + 강한 진동 | 경고 패턴 |

구현: `navigator.vibrate()` (Web Vibration API, Android 지원)
iOS: 제한적 지원 → `useHaptic` 훅에서 플랫폼 감지 후 노-옵(no-op)

### 3.7 접근성 인터랙션 패턴

**키보드 네비게이션:**
- 탭 바: `role="tablist"`, `role="tab"`, 화살표 키 이동
- 바텀시트: `role="dialog"`, `aria-modal="true"`, ESC 닫기
- 채팅 입력: Enter 전송, Shift+Enter 줄바꿈

**스크린 리더 지원:**
- 사주 기둥 테이블: `<table>`, `<caption>`, `scope` 속성
- 오행 차트: `role="img"`, `aria-label="오행 분포: 목 2개, 화 3개..."` 대체 텍스트
- 스트리밍 텍스트: `aria-live="polite"` (토큰 추가 시 자동 낭독)
- 로딩 상태: `aria-busy="true"`, `aria-label="사주 분석 중"`

**색상 대비 기준 (WCAG 2.1 AA):**

| 색상 조합 | 대비율 | 기준 충족 |
|-----------|--------|-----------|
| 본문 텍스트(#2C2C2C) on 배경(#FFFFFF) | 15.8:1 | AAA |
| 골드 버튼(#C9A96E) on 네이비(#1A1A2E) | 4.8:1 | AA |
| 보조 텍스트(#8B8B8B) on 백색 | 3.5:1 | AA (큰 텍스트) |
| 오행 컬러 on 라이트 배경 (최소값) | 4.5:1 이상 확인 필요 | AA 목표 |
